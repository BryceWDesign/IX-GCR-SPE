# Security Policy

## Scope

This repository is not an operational flight system and does not deploy
production services. Security issues here generally relate to:
- accidental inclusion of sensitive data,
- unsafe dependency additions,
- or documentation that could be mistaken for qualified flight guidance.

## Reporting

Please report security-relevant concerns by opening a private channel where
possible before public disclosure.

## What to report

- accidental credentials or tokens
- proprietary or non-public material included by mistake
- copied third-party code with unclear licensing
- unsafe instructions that should be reframed or removed
