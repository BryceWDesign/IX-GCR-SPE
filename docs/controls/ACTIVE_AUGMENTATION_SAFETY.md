# Active Augmentation Safety

IX-GCR-SPE keeps active magnetic augmentation as an optional local branch only.

## 1. Purpose
Provide bounded charged-particle deflection around the Shadow Vault while
preserving passive survivability if the field is unavailable.

## 2. Why this branch is bounded
Public studies show that active magnetic shielding remains burdened by:
- enormous mass,
- stored energy,
- structural loads,
- quench management,
- restart complexity.

Therefore, IX-GCR-SPE does not make vehicle-wide active shielding the baseline.

## 3. Safety requirements

### 3.1 Passive independence
The crew vault must remain passively survivable if the active field is absent.

### 3.2 Quench awareness
The branch must include:
- quench detection,
- safe shutdown logic,
- thermal release management,
- crew notification.

### 3.3 EMI/EMC discipline
The branch must consider:
- interaction with dosimeters,
- avionics exposure,
- human factors and local equipment compatibility.

### 3.4 Stored-energy honesty
Any branch that stores significant energy must expose:
- energy inventory,
- safe discharge logic,
- post-failure recovery posture.

## 4. Claim boundary
The local magnetic branch may be described only as an augmentation study,
not as a guaranteed protective shell.

## 5. Preferred use
Use the active branch only around the Shadow Vault or a similarly small, highly
occupied refuge volume where the mass and complexity can be justified.
