# Source Traceability Map

Retrieval dates reflect the public web as checked for this repo build on 2026-04-14.

## S-01 — OLTARIS capability
**Claim used:** OLTARIS is an appropriate public-facing tool for assessing space
radiation effects on shielding materials, electronics, and biological systems.

**Source:**
- NASA OLTARIS home page  
  https://oltaris.nasa.gov/

## Inference notes

The following repo choices are engineering inferences assembled from multiple
sources rather than verbatim source conclusions:
- concentrating mass around a small Shadow Vault instead of uniformly around a
  large habitable volume,
- using AHIS-style shield health monitoring to guard fill state and weak zones,
- keeping active augmentation local rather than vehicle-wide.
