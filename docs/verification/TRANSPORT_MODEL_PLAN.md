# Transport Model Plan

## Primary tools
- **OLTARIS** for rapid initial trade-study screening
- **Geant4** for geometry-explicit mixed-field and secondary-particle analysis

## Baseline comparison set

1. Uniform low-Z wall baseline
2. Water-enhanced local Shadow Vault
3. PE/UHMWPE-heavy Shadow Vault
4. Hydride-enhanced Shadow Vault
5. Cryogenic hydrogen annular Shadow Vault
6. Passive Shadow Vault plus local magnetic augmentation

## Core scenarios

### Scenario A — GCR free-space transit
Purpose:
- compare chronic dose reduction,
- identify diminishing returns,
- inspect mixed-field behavior behind each stack.

### Scenario B — Reference SPE shelter event
Purpose:
- compare acute-event survivability,
- compare added water-equivalent and occupancy posture,
- estimate required vault timing.

### Scenario C — Degraded fill state
Purpose:
- understand sensitivity to partial chamber depletion and leaks.

### Scenario D — Hatch-path weakness
Purpose:
- compare hatch cover thickness and geometry effects.

### Scenario E — Directional / attitude sensitivity
Purpose:
- understand how asymmetric mass placement changes local exposure.

## Outputs to track

- effective dose or dose equivalent where appropriate
- organ-relevant or phantom-based outputs when available
- energy deposition only as a support metric, not the sole performance claim
- secondary-particle spectra
- depth / areal-density response
- uncertainty ranges
- field-on versus field-off deltas for active branch

## Geometry rule
Start simple, but do not hide the hatch, corners, or standoff structure forever.
Those weak zones eventually have to enter the model.
