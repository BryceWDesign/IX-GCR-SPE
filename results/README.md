# Results

This directory is reserved for future outputs such as:
- zone accounting reports,
- OLTARIS screening summaries,
- Geant4 comparison tables,
- degraded-state scenario logs.

## Rules
1. Record assumptions with every result package.
2. Do not post dose plots without geometry and material metadata.
3. Separate:
   - literature values,
   - model outputs,
   - illustrative placeholders.
