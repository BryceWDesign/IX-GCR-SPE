# Contributing

Thank you for contributing to IX-GCR-SPE.

## Ground rules

1. Do not add unsupported protection claims.
2. Mark any speculative branch as **Hypothesis** or **Trade Study**.
3. Keep design claims tied to:
   - source references,
   - measurable inputs,
   - stated assumptions,
   - stated failure modes.
4. Do not copy patented claim language into implementation guidance.
5. Do not present shielding performance using absorbed dose alone when
   dose equivalent or organ-specific metrics are more appropriate.
6. Keep all BOM values explicitly labeled as:
   - measured,
   - literature-derived,
   - or illustrative/notional.

## Pull request expectations

A contribution should include:
- changed files list,
- rationale,
- claim impact,
- source traceability update if technical assertions changed,
- risk register update if a new subsystem was added.

## What not to submit

- miracle material claims
- unsupported electrostatic / resonance / scalar shield claims
- hidden assumptions about 100 percent protection
- code or text copied from non-permissive sources
